# taktakk

**Offline learning for survival, dignity, and self-reliance.**

taktakk delivers education to communities where internet access is
unreliable, power is intermittent, and carrying an educational app can
attract unwanted attention.

The app disguises itself as a plain clock. A specific gesture unlocks
access to a local learning platform. If the device is threatened, all
data can be destroyed in under a second.
